Q-Learning with PacMan:
    IdentityExtractor uses coordinates as state:
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=IdentityExtractor -x 50 -n 60 -l smallGrid -g DirectionalGhost
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=IdentityExtractor -x 500 -n 510 -l smallGrid -g DirectionalGhost
    
    A larger grid makes thing alot harder for the IdentityExtractor.
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=IdentityExtractor -x 50 -n 60 -l mediumGrid -g DirectionalGhost
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=IdentityExtractor -x 1000 -n 1010 -l mediumGrid -g DirectionalGhost
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=SimpleExtractor -x 5 -n 15 -l mediumGrid -g DirectionalGhost
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=SimpleExtractor -x 0 -n 10 -l mediumGrid -g DirectionalGhost

    On a much larger grid
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=SimpleExtractor -x 50 -n 60 -l mediumClassic -g DirectionalGhost
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=SimpleExtractor -x 0 -n 10 -l mediumClassic -g DirectionalGhost -q
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=SimpleExtractor -x 5 -n 15 -l mediumClassic -g DirectionalGhost -q
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=SimpleExtractor -x 50 -n 60 -l mediumClassic -g DirectionalGhost -q
        python pacman.py -z 2.5 -p ApproximateQAgent -a extractor=SimpleExtractor -x 100 -n 110 -l mediumClassic -g DirectionalGhost -q
